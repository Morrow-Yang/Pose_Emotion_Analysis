## 1. 数据集概述

### 1.1 BVH 动作捕捉数据集

| 属性 | 值 |
|------|----|
| 文件格式 | BVH（Biovision Hierarchy，关节角度） |
| 总样本数 | **1,402** 个动作片段 |
| 帧率 | **125 FPS** |
| 片段时长 | 6.2–15.9 秒（P50 = 8.1 秒，P90 = 9.3 秒） |
| 情绪类别 | 7 类（Angry, Disgust, Fearful, Happy, Neutral, Sad, Surprise） |
| 骨骼 | 60+ 关节，層级正向运动学（FK） |
| 坐标类型 | 关节局部旋转角 → 需要 FK 计算世界坐标 |

**数据特点**：BVH 文件以关节旋转角度（欧拉角或四元数）编码运动，需要通过正向运动学（Forward Kinematics）将层级旋转转换为世界坐标，才能计算关节位置和速度。片段时长可变（6–16秒），增加了跨样本比较的难度。

### 1.2 EBM 数据集（Emotional Body Motion）

| 属性 | 值 |
|------|----|
| 文件格式 | CSV（世界坐标，单位：米） |

| 采集结构 | 29 演员 × 4 场景 × 5 次重复 × 7 情绪 = 4,060 |
| 帧率 | **~30 FPS** |
| 片段时长 | **固定 150 帧 = 5 秒**（所有样本等长） |
| 情绪类别 | 7 类（各类 **580** 个样本，完全均衡） |
| 关节数 | **19 个**关节（躯干、四肢、头部） |

**数据特点**：EBM 直接提供世界坐标，无需复杂的运动学计算。固定150帧的统一长度为时间分段分析提供了天然对齐基准。样本完全均衡（无类别不平衡问题）。29 名演员覆盖了较多个体差异。

## 2. 分析框架

两个数据集共享相同的分析流水线，以保证结果可比性：

```
原始数据（BVH / EBM CSV）
        ↓
  特征提取（几何 + 速度 + 能量 + [时间动态]）
        ↓
  ┌─────────────────────────────────────────────────────┐
  │  Kruskal-Wallis  │  PCA / t-SNE  │  随机森林分类器  │
  │  情绪区分能力     │  高维可视化   │  7 类情绪识别    │
  └─────────────────────────────────────────────────────┘
        ↓
  情绪间差异解读 + 时间动态分析（EBM）
```

**统计框架选择原则**：
- 所有特征分布均可能偏态（速度/能量高度右偏），使用 Kruskal-Wallis 非参数检验而非 ANOVA
- 效应量 $\varepsilon^2 = (H - k + 1) / (N - k)$，遵循 Cohen（1988）阈值：大效应 ≥ 0.14，中效应 0.06–0.14，小效应 0.01–0.06
- RF 分类器使用 **GroupShuffleSplit 按演员分组**划分训练/测试集，防止同一演员数据泄露至测试集（避免身份识别替代情感识别）

## 3. 特征提取方法

### 3.1 几何特征（Geometrical Features）

几何特征描述**某时刻身体的空间构型**，与运动速度无关，捕捉的是"摆什么姿势"。

所有几何计算在**归一化坐标系**下进行：以 Hips 为原点，以 Head-Hips 连线长度（脊柱高度）为单位长度缩放，消除演员身高差异。

| 特征 | 计算方式 | 物理含义 |
|------|---------|---------|
| `arm_span_norm` | `‖LeftHand - RightHand ‖ / spine_len` | 双手展开宽度（归一化） |
| `left/right_elbow_angle` | 以前臂为顶点的三点夹角 | 肘关节弯曲程度 |
| `left/right_knee_angle` | 以膝盖为顶点的三点夹角 | 膝关节弯曲程度 |
| `trunk_tilt_deg` | Hips→Shoulder 向量与垂直轴夹角 | 上半身前后倾斜角 |
| `spine_bend_deg` | Hips→Spine→Head（或 Spine1）链的总弯曲 | 脊柱弯曲程度 |
| `lateral_lean_deg` | 躯干向左/右偏移角度 | 侧向倾斜 |
| `head_dz` | Head 相对 Hips 在深度方向（z）的归一化位移 | 头部前伸/后缩 |
| `head_dy` | Head 相对 Hips 在上下方向（y）的归一化位移 | 头部高度变化 |
| `left/right_hand_height` | 手部在 y 轴的归一化高度 | 抬手高度 |
| `shoulder_width` | 双肩连线长度（归一化） | 肩部张开程度 |
| `body_extent` | 所有关节构成的 3D 包围盒对角线长度 | 整体姿态张开度 |
| `contraction_index` | 所有肢端到躯干中心的平均距离（取负）| 全身收缩/舒展程度 |
| `elbow_asym` | 左右肘角差的绝对值 | 左右不对称性 |
| `pelvis_height_norm` | Hips 相对脚底的归一化高度 | 重心高低（蹲/站） |

每个几何量对每帧提取一次，再对整段 clip 计算 **mean / std / range**，得到聚合特征。BVH 每5帧采样一次（减少计算量），EBM 每5帧采样一次。

BVH 共提取 **72 个几何特征**；EBM 共提取 **75 个**（坐标轴定义略不同，无 Spine2/Spine3，用 Spine1 替代）。

### 3.2 速度特征（Velocity Features）

速度特征描述**关节移动的快慢**，捕捉的是"动得多快"。

对关键关节逐帧计算速度：

$$v_j(t) = \|p_j(t) - p_j(t-1)\| \times \text{FPS} \quad \text{(m/s)}$$

选取关节（根据运动学意义选择信息量最大的关节）：

| 关节 | BVH 对应 | EBM 对应 | 意义 |
|------|---------|---------|------|
| 头部 | Head | Head | 头部摆动/点头 |
| 手腕（左/右）| LeftHand / RightHand | LeftHand / RightHand | 手臂末端速度（最大）|
| 肘部（左/右）| LeftForeArm / RightForeArm | LeftForeArm / RightForeArm | 前臂速度 |
| 肩部（左/右）| LeftArm / RightArm | LeftArm / RightArm | 大臂近端速度 |
| 全身平均 | 以上各关节均值 | 以上各关节均值 | 综合运动强度 |

对每个关节的速度序列提取 **mean / std / max**（range = max-min），共得到 **24 个速度特征**（EBM / BVH 相同）。

### 3.3 动态能量特征（Dynamic Kinetic Energy Features）

能量特征从速度出发，描述**运动的强度模式**，包括绝对强度、变化程度、节律和空间分布。

**瞬时动能**：

$$E(t) = \sum_{j} v_j(t)^2$$

基于 $E(t)$ 时间序列计算 8–11 个能量特征：

| 特征 | 计算方式 | 含义 |
|------|---------|------|
| `E_mean` | $\overline{E(t)}$ | 全段平均动能（整体激活水平）|
| `E_std` | $\sigma(E(t))$ | 动能变异性 |
| `E_max` | $\max E(t)$ | 峰值动能 |
| `E_range` | $E_{max} - E_{min}$ | 动能动态范围 |
| `E_cv` | $\sigma(E) / \overline{E}$ | 变异系数（相对变化幅度）|
| `E_skew` | 偏度 | 能量分布的不对称性 |
| `burst_count` | find_peaks(height=1.5×median, distance=5) | 能量爆发次数 |
| `dom_freq_hz` | FFT 主频 | 运动循环频率（Hz）|
| `arms_share_mean` | 手臂关节能量 / 总能量 | 手臂运动的相对贡献 |
| `head_share_mean` | 头部能量 / 总能量 | 头部运动的相对贡献 |

BVH 额外提取 `E_min` 和 `E_skew`，共 11 个能量特征；EBM 共 8 个。

## 4. BVH 数据集分析结果

### 4.1 几何特征的情绪区分能力（KW 检验）

72 个几何特征的 Kruskal-Wallis 检验结果：

- **大效应（ε² ≥ 0.14）**：26 个
- **中效应（0.06–0.14）**：40 个
- **小效应（<0.06）**：6 个
- **总显著（p < 0.05）**：接近全部

**Top 15 几何特征（按 ε² 排序）**：

| 排名 | 特征 | ε² | 含义 |
|-----|------|-----|------|
| 1 | `left_elbow_angle_range` | **0.208** | 左肘角变化幅度 |
| 2 | `left_elbow_angle_std` | 0.208 | 左肘角标准差 |
| 3 | `left_hand_height_mean` | 0.206 | 左手平均高度 |
| 4 | `contraction_range` | 0.191 | 全身收缩幅度变化 |
| 5 | `pelvis_height_norm_range` | 0.191 | 重心高度变化幅度 |
| 6 | `right_elbow_angle_range` | 0.187 | 右肘角变化幅度 |
| 7 | `left_hand_height_range` | 0.187 | 左手高度变化幅度 |
| 8 | `pelvis_height_norm_std` | 0.186 | 重心高度标准差 |
| 9 | `left_hand_height_std` | 0.183 | 左手高度标准差 |
| 10 | `left_elbow_angle_mean` | 0.182 | 左肘角均值 |
| 11 | `elbow_asym_range` | 0.176 | 左右肘不对称性变化 |
| 12 | `right_elbow_angle_std` | 0.175 | 右肘角标准差 |
| 13 | `right_hand_height_range` | 0.172 | 右手高度变化幅度 |
| 14 | `elbow_asym_mean` | 0.172 | 左右肘不对称均值 |
| 15 | `right_knee_angle_mean` | 0.172 | 右膝角均值 |

**规律**：动态变化量（range/std）普遍优于静态均值（mean）；肘关节是关键的情绪信号载体，尤其是其弯曲幅度的变化范围（`elbow_angle_range`）。

### 4.2 动态能量特征的情绪区分能力

11 个能量特征的 KW 效应量：

| 特征 | ε² | 等级 | 解读 |
|------|-----|------|------|
| `E_mean` | **0.367** | 大 | 总体激活水平差异极显著 |
| `E_std` | 0.357 | 大 | 能量变异度差异极显著 |
| `E_max` | 0.302 | 大 | 峰值能量差异极显著 |
| `E_range` | 0.302 | 大 | 动态范围差异极显著 |
| `arms_share_mean` | 0.125 | 中 | 手臂能量贡献有差异 |
| `head_share_mean` | 0.114 | 中 | 头部能量贡献有差异 |
| `burst_count` | 0.100 | 中 | 爆发次数有差异 |
| `E_cv` | 0.099 | 中 | 相对变化幅度有差异 |
| `E_skew` | 0.073 | 中 | 能量分布偏态有差异 |
| `dom_freq_hz` | 0.053 | 小 | 主运动频率差异较小 |
| `E_min` | 0.015 | 小 | 最低能量区分力弱 |

**关键发现**：`E_mean` 的 ε²=0.367 是全体几何和速度特征中最高的效应量之一（几何最高为 0.208），说明总体运动强度水平是最强的情绪区分维度。

**各情绪能量画像（BVH，中位数）**：

| 情绪 | E_mean | E_cv | burst_count | dom_freq（Hz） | arms_share |
|------|--------|------|------------|--------------|-----------|
| **Happy** | **0.0012**（最高）| 1.631 | 46 | 0.19 | — |
| Angry | 0.0008 | **2.014**（最高）| 48 | 0.18（最慢）| — |
| Fearful | 0.0005 | 1.667 | 48 | 0.27 | — |
| Surprise | 0.0003 | 1.687 | 46（最少）| 0.39 | — |
| Disgust | 0.0003 | 1.574 | 51 | 0.23 | — |
| Sad | **0.0002**（最低）| 1.913 | **57**（最多）| 0.35 | — |
| **Neutral** | 0.0002（最低）| **1.310**（最低）| 49 | **0.48**（最快）| — |

**情绪能量规律**：
- **Happy** 动能最高，体现快乐带来的全身活跃程度
- **Neutral** 整体能量最低且最均匀（E_cv 最低），呈现规律、稳定的中性动作
- **Angry** E_cv 最高（2.014），说明愤怒的能量波动最剧烈——有强烈的爆发，也有短暂的停顿
- **Sad** 爆发次数最多（57次），但 E_mean 最低——说明悲伤并非全程低沉，而是有频繁的微小起伏
- **Neutral** 主频最高（0.48 Hz），对应规律、有节奏的日常动作；**Angry** 主频最低（0.18 Hz），对应缓慢积蓄后爆发的模式

### 4.3 随机森林分类结果

分类器：300 棵随机森林，按演员 GroupShuffleSplit 分组，test_size=0.2  
特征集：几何（72）+ 速度（24）共 96 个特征

- **整体准确率：58.25%**（Baseline 随机猜测 = 14.3%，提升约 4.1×）
- Neutral 类表现最佳（高精确率），高激活情绪（Happy/Angry）区分较好

**RF 重要性前 10 特征（BVH）**：以速度类和肘关节几何类为主，与 KW 分析一致。

## 5. EBM 数据集分析结果

### 5.1 特征体系

EBM 共提取 **107 个特征**（每文件聚合为一行）：

| 类别 | 特征数 | 代表特征 |
|------|--------|---------|
| 速度特征 | 24 | 各关节速度的 mean/std/max |
| 能量特征 | 8 | E_mean, E_cv, dom_freq, arms_share, head_share 等 |
| 几何特征 | 75 | 臂展、肘/膝角、躯干倾斜、手部高度等的 mean/std/range |

### 5.2 Kruskal-Wallis 情绪区分能力

107 个特征的 KW 检验：

- **大效应（ε² ≥ 0.14）**：10 个
- **中效应（0.06–0.14）**：58 个
- **小效应（0.01–0.06）**：32 个
- **可忽略（< 0.01）**：3 个
- **总显著（p < 0.05）**：**104 / 106**（全面显著）

**Top 10 特征（ε² 排序）**：

| 排名 | 特征 | ε² | 类别 |
|-----|------|-----|------|
| 1 | `l_wrist_vel_mean` | **0.211** | 速度 |
| 2 | `r_wrist_vel_mean` | 0.178 | 速度 |
| 3 | `avg_velocity_mean` | 0.166 | 速度 |
| 4 | `E_mean` | 0.160 | 能量 |
| 5 | `l_elbow_vel_mean` | 0.160 | 速度 |
| 6 | `head_share_mean` | 0.151 | 能量 |
| 7 | `l_wrist_vel_std` | 0.150 | 速度 |
| 8 | `l_wrist_vel_max` | 0.145 | 速度 |
| 9 | `arms_share_mean` | 0.145 | 能量 |
| 10 | `r_elbow_vel_mean` | 0.142 | 速度 |

**规律**：大效应特征中 7 个来自速度类，3 个来自能量类，几何特征最高效应量（arm_span_norm_mean）仅约 0.131——说明**运动强度（速度/能量）比姿势构型（几何）对情绪的区分力更强**。

**按类别统计大效应特征数**：速度类（7/10 大效应）> 能量类（3/10）> 几何类（0 大效应，但 58 个中效应）

### 5.3 各情绪特征画像（EBM，中位数）

| 情绪 | E_mean | avg_vel | arm_span | 肘角 | 躯干倾斜 | head_share | arms_share |
|------|--------|---------|---------|------|---------|-----------|-----------|
| **Angry** | **3.89**（最高）| **0.517** | 5.17 | 101° | 5.9° | 5.3% | **84.0%** |
| **Happy** | 2.31 | 0.363 | **5.74**（最宽）| **113°** | 5.9° | 5.7% | 83.9% |
| Fearful | 1.53 | 0.294 | 4.45 | 100° | 7.1° | **8.9%**（最高）| 79.6% |
| Neutral | 1.29 | 0.306 | 3.92 | 103° | 6.9° | 8.6% | 80.1% |
| Sad | 0.91 | 0.228 | **2.93**（最窄）| **83°**（最弯）| 9.4° | 9.9% | 78.6% |
| Disgust | 0.72 | 0.184 | 3.23 | 100° | **9.5°**（最斜）| 10.8%（最高）| 78.2% |
| **Surprise** | **0.68**（最低）| 0.201 | 5.31 | **139°**（最直）| **4.7°**（最正）| 9.2% | 79.7% |

**情绪画像解读**：

- **Angry（愤怒）**：运动能量与速度全体最高，手臂能量贡献最大（84%）。体现全身有力的攻击性动作，大幅度快速挥臂。

- **Happy（快乐）**：运动能量第二高，臂展最宽（5.74），肘关节较伸展（113°）。身体以开放、舒展的姿态体现喜悦，双臂自然张开。

- **Surprise（惊讶）**：运动能量最低（0.68），但臂展很大（5.31）、肘角最直（139°）、躯干最正（4.7°）。**这是一个"静态情绪"**：并非通过大量运动表达，而是在一瞬间切换到特定的张臂/挺身/屏气姿势，之后保持相对静止。

- **Fearful（恐惧）**：能量中等，头部能量占比显著（8.9%）——头部的防御性动作（缩头、转头）是恐惧的独特信号。

- **Sad（悲伤）**：臂展最窄（2.93），肘角最弯（83°），躯干后倾（9.4°）。身体内收、双臂贴紧身体，呈现经典的"蜷缩"悲伤姿态。

- **Disgust（厌恶）**：躯干倾斜最大（9.5°），运动能量低，呈现"上半身后仰回避"的典型厌恶姿态。

### 5.4 PCA / t-SNE 降维分析

PCA 第一主成分（PC1）与能量维度高度对应：Angry、Happy 投影在高端，Sad、Disgust 在低端。第二主成分（PC2）捕捉了部分几何信息（体型舒展度 vs 收缩度）。

t-SNE 显示局部聚集效应：同情绪样本形成松散的邻域簇，但情绪间边界模糊——反映情绪表达的连续性本质以及个体差异（同一演员的多次录制紧密聚集，跨演员分布更分散）。

### 5.5 随机森林分类结果

分类器：300 棵随机森林，按演员 GroupShuffleSplit，test_size=0.2  
特征集：全部 107 个特征

- **整体准确率：49.9%**（Baseline = 14.3%，提升 3.5×）
- **宏平均 F1：0.497**
- **与人类准确率（48.4%）基本持平**

**各情绪分类指标**（测试集 n=120/类）：

| 情绪 | 精确率 | 召回率 | F1 |
|------|--------|--------|-----|
| **Angry** | 0.690 | 0.483 | **0.569**（最佳）|
| Surprise | 0.449 | **0.667** | 0.537 |
| Disgust | 0.446 | 0.650 | 0.529 |
| Neutral | 0.459 | 0.558 | 0.504 |
| Happy | **0.706** | 0.400 | 0.511 |

## 6. 时间动态分析（EBM）

EBM 固定150帧的特性使其成为时间分析的理想数据集。

### 6.1 动机

传统聚合特征（mean/std/max）将每段5秒动作压缩为少数数字，丢失了"能量如何随时间演变"的信息。本分析追问：**同一种情绪，其运动在5秒内是如何展开的？**

### 6.2 特征设计

#### A. 时间分段特征（Phase Features，15个）

将149帧速度序列等分为**5段**（每段约30帧 = 1秒），对每段计算：
- `E_phase{1-5}`：平均动能（绝对强度）
- `vel_phase{1-5}`：平均关节速度（绝对强度）
- `E_ratio_phase{1-5}`：该段能量占总能量比例（相对分布）

#### B. 时间形状描述符（Shape Descriptors，7个）

将整条149帧曲线视为一个整体，提取其"形态特征"：

| 特征 | 公式/方法 | 含义 |
|------|---------|------|
| `peak_time_norm` | $\arg\max E(t) / 148$ | 能量峰值时刻（0=开头，1=结尾）|
| `energy_front_ratio` | $\sum_{t<T/2} E(t) / \sum E(t)$ | 前半段能量占比 |
| `rise_time_norm` | 首次达到80%峰值的帧数 / 148 | 起爆速度 |
| `sustain_ratio` | $P\{E(t) \geq 0.5 E_{max}\}$ | 高强度持续比例 |
| `energy_slope` | $\text{polyfit slope} / \overline{E}$ | 能量整体趋势（归一化）|
| `energy_autocorr_lag1` | 滞后1帧自相关 | 运动平滑度 |
| `jerk_mean` | $\text{mean}(|vel(t) - vel(t-1)| \times 30)$ | 速度变化急促感 |

> `jerk_mean` 是本分析中**唯一**原始静态特征集完全未涵盖的物理量。

### 6.3 Kruskal-Wallis 结果

22 个时间特征：**7 大效应，4 中效应，10 小效应**

| 特征 | ε² | 等级 |
|------|-----|------|
| `jerk_mean` | **0.204** | 大 |
| `vel_phase4` | 0.157 | 大 |
| `vel_phase5` | 0.155 | 大 |
| `E_phase4` | 0.154 | 大 |
| `E_phase5` | 0.154 | 大 |
| `E_phase3` | 0.151 | 大 |
| `vel_phase3` | 0.147 | 大 |
| `E_phase2` | 0.130 | 中 |
| `vel_phase2` | 0.128 | 中 |
| `E_phase1` | 0.116 | 中 |
| `vel_phase1` | 0.106 | 中 |
| `energy_slope` | 0.052 | 小 |

**关键发现 1**：Phase 越晚，区分力越强。Phase5（ε²=0.154）> Phase4（0.154）> Phase3（0.151）> Phase2（0.130）> Phase1（0.116）。情绪特征在动作展开过程中逐渐显现，后半段（2–5秒）的运动模式比前半段更具情绪特异性。

**关键发现 2**：`jerk_mean`（加加速度）ε²=0.204，与原有最优特征 `l_wrist_vel_mean`（0.211）几乎持平，是一个静态分析完全遗漏的高价值物理量。

### 6.5 三种时间动态模式

基于相位能量分布与形状描述符，7种情绪呈现三类时间动态原型：

---

**模式 A：前倾急衰型（Front-Loaded, Fast-Decay）**

**对应情绪：Fearful、Sad、Disgust**

特征：前半段能量占比 **65–70%**，Ph1/Ph5 比率 **3.8–5.9×**，能量斜率 **<−1.0**，峰值时刻出现在 **0.17–0.26**（前1/4段）。

```
能量  █████░░░░  高  →  迅速下降  →  低平  →  低平  →  低平
         0-1s     1-2s    2-3s    3-4s    4-5s
```

三种情绪的区别：Fearful 的 Phase3–5 几乎形成"平台"（0.70→0.58→0.59，趋于稳定低位，即"僵住"）；Disgust 持续单调下降到最低（0.94→0.16，无平台）；Sad 下降速率居中，后期缓慢消散。

**心理学解释**：这三种均属于"负向回避情绪"（negative/avoidance emotions）。运动模式符合**趋避反应（approach-avoidance response）**：遭遇威胁/厌恶刺激后本能应激（Phase1高能量），随后快速转入抑制/回避状态。Fearful 的后期平台可能对应"冻结反应（freezing response）"。

---

**模式 B：持续维持型（Sustained, Stable）**

**对应情绪：Angry、Happy**

特征：Ph1/Ph5 比率仅 **1.3–1.5×**，能量斜率 **>−0.40**（近乎水平），Jerk 值最高（**1.18–1.92**）。

```
能量  ██████████  高  ~  高  ~  中高  ~  中高  ~  中
         0-1s     1-2s    2-3s    3-4s    4-5s
```

Angry：能量在 Phase2 达峰（3.44），之后极缓慢下降（斜率−0.10），5秒结束时仍高度活跃（2.54）。Jerk=1.922，是 Surprise 的 3.4×——愤怒不是平滑流畅的大动作，而是充满急促的加速减速。

Happy：独特之处在于 **Phase3 出现回升**（1.58→1.73），形成"中段小高潮"，其他情绪均无此特征。可能对应大笑、跳跃、鼓掌等节律性高兴动作的中段激活。

**心理学解释**：这两种均属于"高激活/趋近情绪（high-arousal/approach emotions）"。持续的情绪动机驱动持续性动作输出，不存在快速消退模式。

---

**模式 C：平稳低调型（Flat, Restrained）**

**对应情绪：Surprise、Neutral**

特征：Jerk 最低（**0.56–0.85**），自相关最高（**≥ 0.95-0.96**），全程低能量。

```
能量  ████░░░░░  中低  ~  低  ~  低  ~  低  ~  低
         0-1s     1-2s    2-3s    3-4s    4-5s
```

Surprise 的 Phase2–3 形成短暂能量平台（0.45→0.45），平滑度极高——运动极度流畅，几乎无任何突发加速。这与 Surprise 在静态几何特征中呈现的极端姿态（臂展5.31，肘角139°，躯干最正4.7°）形成对照：**Surprise 的信息量集中在姿态变化（几何），而非运动强度（动能）**。

**心理学解释**：Surprise 对应"定向/冻结反应（orienting response）"——惊讶时身体暂停动作以全力处理新信息，这是一种进化上的本能。Neutral 则是正常活动的基线，无特定情绪驱动。

### 6.6 RF 对比实验

| 特征集 | 特征数 | 准确率 | Macro-F1 |
|--------|--------|--------|---------|
| 基线（静态）| 107 | 43.9% ± 3.5% | 0.433 ± 0.039 |
| 基线 + 时间动态 | 129 | 43.1% ± 2.8% | 0.425 ± 0.031 |

加入22个时间特征后准确率基本不变（Δ=−0.8%）。原因：`E_phase{1-5}` 与 `E_mean/E_cv` 存在信息冗余——均描述能量分布，只是粒度不同。

这并不否定时间分析的价值：时间特征在**解释性研究**（理解情绪的运动机制）中具有独立意义，而在**预测性研究**（提高分类准确率）中需要与静态特征更精细地去线性相关。真正具有增量信息的特征是 `jerk_mean`，但其单独贡献难以在整体准确率上体现。

## 7. 跨数据集比较

### 7.1 一致性发现

以下结论在 BVH 和 EBM 两个数据集中均得到支持：

1. **手臂/手腕运动是情绪识别最重要的信号来源**。BVH 中肘关节几何特征 ε²~0.20，EBM 中腕部速度 ε²~0.21，这一结论跨数据集高度稳定。

2. **速度/能量特征的区分力强于几何/姿态特征**。两个数据集中，绝对运动强度（E_mean）均名列前茅；EBM 中大效应特征 100% 来自速度/能量类，几何类尽管中效应丰富但无大效应。

3. **Happy 或高激活情绪能量最高，Sad 最低**。BVH 中 Happy E_mean 最高，Sad 最低；EBM 中 Angry 能量最高（BVH 未排第一可能受录制内容影响），Surprise/Disgust 最低。两数据集均支持"激活维度（arousal）是情绪运动的第一主成分"。

4. **Neutral 区分最难**。两数据集中 Neutral 均无明显极端特征，是混淆的主要来源。

### 7.2 差异对比

| 维度 | BVH 特点 | EBM 特点 | 解释 |
|------|---------|---------|------|
| 准确率 | **58.3%** | 49.9% | BVH 样本更多特征更丰富；EBM 演员更多个体差异更大 |
| 情绪区分最强特征 | 肘角变化幅度（几何）| 腕部速度（速度）| BVH 通过 FK 角度直接描述关节运动学，几何特征更精确 |
| 最易识别情绪 | Neutral（高精确率）| Angry（F1=0.569）| 录制场景/演员差异 |
| 固有优势 | 高时间分辨率（125Hz）| 等长片段（5秒），世界坐标 | 不同适用场景 |

### 7.3 方法论差异说明

BVH 需要正向运动学将关节旋转角转换为世界坐标，引入了 FK 计算误差；EBM 直接提供世界坐标，精度更高但关节数较少（19 vs 60+）。两者在几何特征上有较大重叠（肘角、手部高度、臂展均可提取），速度特征计算方法完全相同。

## 8. 主要发现汇总

### 跨数据集一致性结论

1. **手臂/腕部运动速度是区分情绪的最强信号**（BVH: ε²~0.208，EBM: ε²~0.211），在两个独立数据集中汇聚形成跨数据集支持。

2. **情绪识别的物理核心是运动强度（激活维度）**，而非身体构型（几何维度）。主成分分析和效应量排名一致指向 E_mean 和关节速度作为第一判别维度。

3. **机器学习分类器（49.9%–58.3%）达到人类识别水平（≈48%）**，验证了特征体系的有效性，同时揭示7类情绪识别的固有困难性。

### 情绪特异性发现

| 情绪 | 最独特的运动特征 | 数据支持 |
|------|--------------|---------|
| **Angry 愤怒** | 最高运动能量（EBM E_mean=3.89），最高动作急促感（jerk=1.92 ），能量持续维持至片段末尾（斜率−0.10）| EBM 能量 + 时间分析 |
| **Sad 悲伤** | 最窄臂展（2.93），最弯肘角（83°），前倾急衰型时间模式，Burst 频率高但单次能量低 | EBM 几何 + BVH 能量 + 时间分析 |
| **Surprise 惊讶** | "静态情绪"：最低运动能量（EBM:0.68）但最直肘角（139°）、最宽广姿态——用姿势而非运动表达情绪 | EBM 几何 + 时间分析 |
| **Happy 快乐** | 最宽臂展（5.74），Phase3 能量回升（中段激活的独特模式），BVH 中能量最高 | BVH+EBM 能量 + 时间分析 |
| **Fearful 恐惧** | 最早能量峰值（0.17），前倾急衰后出现"凝固平台"（后三段能量稳定在低位），头部能量占比高（保护性动作）| EBM 时间分析 + 能量特征 |
| **Disgust 厌恶** | 最大躯干后倾（9.5°），最单调持续的能量衰减（无平台，5.9× 首尾比），动作流畅（高自相关）| EBM 几何 + 时间分析 |
| **Neutral 中性** | 各项指标居中，BVH 中主运动频率最高（0.48Hz，均匀有节奏），是混淆的中心 | BVH 能量 |

### 时间动态核心发现

1. **情绪的运动特征在时间轴后半段（2–5秒）比前半段（0–2秒）更具区分力**（Phase5 ε²=0.154 > Phase1 ε²=0.116），与常识预期（"第一反应最独特"）相悖，揭示情绪运动模式是**逐渐展开**而非立即呈现的。

2. **动作"急促感"（jerk_mean）是完全独立的新信息维度**（ε²=0.204），无法被传统聚合特征替代，且与情绪激活程度高度相关（Angry 1.92 vs. Surprise 0.56）。

3. **三类时间原型与情绪心理学的趋近-回避框架高度对应**：前倾急衰型（回避情绪）、持续维持型（趋近情绪）、平稳低调型（冻结/定向）。
